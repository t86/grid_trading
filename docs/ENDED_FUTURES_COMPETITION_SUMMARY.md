# 已结束合约交易赛固定汇总

口径：

- `最终门槛`：该奖励段末位名次在完赛时的榜单门槛。
- `奖励折USDT`：单个用户在该奖励段可获得的奖励，按完赛时币种价格折算成 USDT。
- `奖励 / 门槛`：`奖励折USDT ÷ 最终门槛`。
- `BIRB / ELSA / MAGMA / COLLECT` 这 4 个活动，当前未拿到可用的完赛 Binance 历史收盘价，所以 `奖励折USDT` 和 `奖励 / 门槛` 暂时留空。
